K compiler

轻量级脚本语言

K语言 是一款仅支持整数类型的脚本语言，内置变量、数组、表达式求值（遵循C++优先级）及完整的流程控制（if/else/repeat/while/break/continue）。其设计目标是为快速编写算法和小工具提供极简环境。

核心特性：

  • 仅整数类型，变量/数组/表达式
  • 严格的空格规则（赋值、索引、表达式无空格）
  • 内置 print/input/color/wait/system/error 等命令
  • 支持行首注释 #

K语言 教程

K语言是一款轻量级脚本语言，仅支持整数类型，内置变量、数组及流程控制，适合快速编写算法和小工具。本教程完全基于解释器实际行为编写，请严格遵守规则，否则可能导致语法错误。

🚀 快速开始

获取解释器

  • 直接运行 K_compiler.exe，或自行编译（Dev-C++ 打开项目编译即可）。
  • 脚本文件必须使用 .kco 扩展名。

运行你的第一个程序

  1. 新建文本文件 hello.kco，输入以下内容（严格注意空格规则）：
a=3
b=5
c=a+b
print c
finish
  1. 将文件拖拽到 K_compiler.exe 上，或双击 exe 后选择“是”并输入文件路径。
  2. 屏幕上将输出 8。

⚠️ 空格规则（非常重要）

解释器对空格的要求极其严格，请务必遵守：

  1. 赋值语句中不能有任何空格

✅ 正确：x=5 arr[0]=10

❌ 错误：x = 5 arr[0] = 10

  1. 数组索引中不能有任何空格

✅ 正确：arr[0] arr[i+1]

❌ 错误：arr[ 0 ] arr[i + 1]

  1. 表达式中不能有任何空格

运算符和操作数之间不能有空格，整个表达式必须紧密相连。

✅ 正确：(3+5)*2 a>10&&b<20

❌ 错误：(3 + 5) * 2 a > 10 && b < 20

  1. 命令与参数之间可以有空格

例如 print 123 会输出 123（前导空格保留）。参数内部的空格也会原样输出。

✅ 正确：print Hello → 输出 Hello

✅ 正确：print World → 输出 World（两个前导空格）

  1. 所有 end 关键字必须单独占一行，前后不能有空格

✅ 正确：

if x>0
if end

❌ 错误：

if x>0
  if end    # 前面有空格

📖 核心语法

注释

仅支持行首注释：以 # 开头的行会被解释器忽略。行末注释不支持。

# 这是一条正确的注释
print 123  # 这是错误的行末注释（不支持）

📦 变量与数组

  • 变量名由字母、数字、下划线组成，不能以数字开头。
  • 所有变量均为整数类型，无法存储字符串。
  • 支持数组，索引从 0 开始，索引值可以是表达式。数组索引中不能有空格。
x=10
arr[0]=5
arr[1]=x+3
print arr[0]+arr[1]

🧮 表达式

支持以下运算符（优先级严格遵循 C++ 惯例，可用 () 改变顺序）。表达式中不能有空格。

类型运算符优先级
乘方^最高
乘除取余* / %较高
加减+ -中等
移位<< >>稍低
比较< > <= >= == !=较低
位与&低
位异或~更低
位或|很低
逻辑与&&极低
逻辑或||最低

示例：

a=(3+5)*2
b=a>10&&a<20
c=16%3
d=2^3

🔁 流程控制

if / else

  • if 后跟表达式，若值非 0 则执行后续语句直到 if end。
  • 可选的 else 单独占一行，后跟假分支语句直到 else end。
score=85
if score>=60
print Pass
if end
else
print Fail
else end

repeat 循环

repeat 后跟次数表达式，循环体以 repeat end 结束。

i=0
repeat 5
i=i+1
print i
repeat end

while 循环

while 后跟条件表达式，条件非 0 时重复执行，以 while end 结束。

i=0
while i<5
print i
i=i+1
while end

循环控制

  • break：立即跳出当前循环。
  • continue：跳过本次循环剩余部分。

📚 内置命令

命令说明示例
print 内容输出内容。若内容是变量名则输出变量值；否则原样输出。print x
input从键盘读入一个整数。x=input
if 表达式条件判断。if x>0
elseif的假分支。
repeat 次数固定次数循环。repeat 10
while 表达式条件循环。while x<100
color 两位十六进制设置控制台背景色。color 0a
wait 毫秒暂停执行。wait 1000
system 命令执行系统命令。system pause
error 提示信息以错误信息终止程序。error 除数不能为零
exit立即终止程序。
continue跳过本次循环剩余部分。
break跳出当前循环。
clear清空控制台屏幕。
finish标记程序正常结束。

注意：所有 end 关键字必须单独占一行，且前后不能有空格。

bool is_allow(char c)
{
    string allow;
    for(char i='0';i<='9';i++)
    {
        allow+=i;
    }
    for(char i='a';i<='z';i++)
    {
        allow+=i;
    }
    for(char i='A';i<='Z';i++)
    {
        allow+=i;
    }
    allow+="`~!@#$%^&*-_=+\\|()[]{};:'\",.<>/?";
    for(int i=0;i<allow.size();i++)
    {
        if(allow[i]==c)
        {
            return true;
        }
    }
    return false;
}

{
