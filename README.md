# kevin1426730.github.io

## Overview

This is the source code for my personal GitHub Pages homepage. It serves as a central hub to showcase my projects and provide quick access to my GitHub profile and hosted web applications.

## Features

- Responsive design that adapts to different screen sizes
- Dark/Light theme toggle with persistent preference storage
- Clean, modern card-based layout
- Quick access links to GitHub profile and homepage
- Project showcase cards with direct links to live demos

## Projects Featured

### K compiler
A lightweight scripting language interpreter that supports integer types, arrays, expression evaluation (following C++ precedence), and complete flow control (if/else/repeat/while/break/continue). The language enforces strict whitespace rules for assignments, array indices, and expressions.
